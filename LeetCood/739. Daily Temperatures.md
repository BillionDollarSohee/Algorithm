[리트코드 739. 일일온도](https://leetcode.com/problems/daily-temperatures/description/)

문제 : 매일의 온도 리스트를 입력받아서, 더 따뜻한 날씨를 위해서는 몇일을 기다려야 하는지를 출력하라. <br>
입력 : [23,24,25,21,19,22,26,23]
출력 : [1,1,4,2,1,1,0,0]
7,8일에는 더이상 따뜻한 날이 없으므로 0이다.

## 자바풀이. 스택으로 빗물트래핑처럼
참고 : [빗물트레핑 문제풀이](https://github.com/BillionDollarSohee/Algorithm/blob/main/LeetCood/42.%20Trapping%20Rain%20Water.md/)

현재의 인덱스를 스택에 쌓아두었다가, 더 높은 온도를 만나면 pop()으로 꺼내고,
현재인덱스와 스택에 쌓아둔 인덱스의 차이로 업데이트 한다.
이후 현재 인덱스를 스택에 담는다.

```java
class Solution {
    public int[] dailyTemperatures(int[] temperatures) {
        int[] result = new int[temperatures.length];

        Deque<Integer> stack = new ArrayDeque<>();
        for (int i = 0; i < temperatures.length; i++) {
            while (!stack.isEmpty() && 
            temperatures[i] > temperatures[stack.peek()]) {
                int last = stack.pop();
                result[last] = i - last;
            }

            stack.push(i);
        }
        return result;
    }
}
```

## 코틀린풀이
한번 선언한 stack변수는 다시 재선언하지 않으므로 모든 변수를 읽기전용 val로 선언하였다.
(실제로 stack의 값은 자유롭게 추가,삭제가 가능하다.)

```kotlin
class Solution {
    fun dailyTemperatures(temperatures: IntArray): IntArray {
        val result = IntArray(temperatures.size)
        val stack = ArrayDeque<Int>()

        for (i in temperatures.indices) {
            while (stack.isNotEmpty() &&
                temperatures[i] > temperatures[stack.last()]) {

                val last = stack.removeLast()
                result[last] = i - last
            }
            stack.addLast(i)
        }

        return result
    }
}
```
